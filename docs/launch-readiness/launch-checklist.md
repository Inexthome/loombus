# Launch Checklist

Use this before promoting Loombus to a larger public audience.

## Product readiness

- Core pages load.
- Signup/login verified.
- Dashboard/member home verified.
- Discussions list verified.
- Discussion detail verified.
- Replies verified.
- Reports verified.
- People/follow/block verified.
- Notifications verified.
- Saved and reading history verified.
- Premium page verified.
- Contact/support verified.

## Legal readiness

- Terms reviewed.
- Privacy reviewed.
- Cookies reviewed.
- Guidelines reviewed.
- Safety reviewed.
- DMCA page reviewed.
- Refund policy reviewed.
- Attorney review scheduled or completed before public scale.
- Trademark plan created.

## Infrastructure readiness

- Supabase backup plan confirmed.
- Vercel deployment rollback known.
- Environment variables checked.
- Admin health checked.
- Admin billing diagnostics checked.
- AI usage dashboard checked.
- Error monitoring plan chosen.
- Analytics plan chosen.
- Rate-limit review completed.

## Billing readiness

- Stripe test checkout verified.
- Stripe webhook verified.
- Stripe live keys not enabled until business/entity setup is ready.
- Refund policy visible.
- Billing portal visible.
- Extra AI Pack fulfillment verified in test mode.
- Admin billing diagnostics visible.

## Support readiness

- Contact form works.
- Admin support queue works.
- Support templates available.
- support@loombus.com visible.
- Safety/reporting paths visible.

## Final smoke commands

- `npm run build`
- live route checks for public pages
- protected API unauthenticated checks
- git status clean
- final checkpoint recorded
