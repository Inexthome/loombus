# Loombus Launch Readiness v1

This folder contains operational runbooks for running Loombus safely before enterprise infrastructure is needed.

Current goal:
- Support public early access and the first 50–100 users.
- Avoid expensive enterprise contracts until usage proves the need.
- Keep infrastructure simple, observable, recoverable, and cost-aware.

Core principle:
Loombus does not need enterprise infrastructure yet. It needs documented backup, monitoring, rate-limit, incident, rollback, and launch procedures.
