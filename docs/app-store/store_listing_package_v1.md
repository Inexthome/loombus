## Apple App Store draft

Subtitle:

Signal over noise

Alternative subtitles:

- Thoughtful discussion
- High-signal conversations
- Discuss with more signal

Keywords draft:

discussion,community,ideas,debate,conversation,network,knowledge,forum,signals

Promotional text:

Loombus helps people create, find, and return to thoughtful discussions built around clarity, context, and useful contribution.

Description:

Loombus is a high-signal discussion platform built for people who want more thoughtful conversations and less noise.

Create focused discussions, explore topic lanes, follow thoughtful contributors, save conversations worth returning to, and use AI-assisted tools to understand key points without derailing the thread.

Loombus is designed around signal over noise:

- Start structured discussions with clear topics and optional tags
- Browse conversations by topic, search, and signal-oriented sorting
- Follow contributors whose ideas, questions, and replies help you think clearly
- Save discussions into a personal reading memory
- Return to alerts for replies, follows, and activity connected to you
- Use Premium AI-assisted tools for summaries, key takeaways, what changed, and disagreement mapping
- Manage your profile, settings, notification comfort, safety, and account controls

Unlike platforms built mainly for endless scrolling, Loombus is built for intentional reading, meaningful replies, and conversations worth revisiting.

Loombus is still growing, but the goal is clear: a cleaner place for thoughtful public discussion.

## Messaging guidance

Use this public positioning first:

Loombus — Signal over noise

Main description:

A high-signal discussion platform for thoughtful conversations.

Avoid using this in the first store listing:

The future of social media

Reason:

That phrase is strong as a founder vision, but the first store listing should be clearer, safer, and more product-specific.
